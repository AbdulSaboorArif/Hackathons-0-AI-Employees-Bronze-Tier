# AI Employee Vault - Bronze Tier

This is your Personal AI Employee's knowledge base and workspace.

## Folder Structure

```
AI_Employee_Vault/
├── Dashboard.md              # Real-time system status and activity
├── Company_Handbook.md       # Rules and guidelines for AI behavior
├── Inbox/                    # Incoming items to be processed
├── Needs_Action/             # Tasks requiring immediate attention
├── Done/                     # Completed tasks archive
├── Plans/                    # Multi-step task plans
├── Logs/                     # Audit logs of all actions
├── Pending_Approval/         # Actions awaiting human approval
├── Approved/                 # Approved actions ready for execution
├── Rejected/                 # Rejected action requests
├── Accounting/               # Financial records and transactions
└── Briefings/                # Generated reports and summaries
```

## Getting Started

1. **Review Company_Handbook.md** - Set your rules and preferences
2. **Check Dashboard.md** - Monitor system status
3. **Start the Watcher** - Run the file system watcher to monitor Inbox
4. **Drop files in Inbox/** - Test the system by adding files
